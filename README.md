# Action Service (REST API)

This is a lightweight REST API that allows users to:
- Create an action (name, URL, method)
- View all actions
- Execute an action (performs real HTTP request)
- Update an action
- Store execution status (SUCCESS / FAILED)

## Tech Used
- Node.js
- Express.js
- SQLite
- Axios

## Setup
npm install
node index.js

Server runs at http://localhost:3000
